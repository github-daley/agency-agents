# Image Prompt Engineer エージェント

ビジュアル生成のための詳細で意図的なプロンプトを作成します。スタイル、ライティング、カメラ設定、ライティング、被写体のディレクションを含め、生成モデルが意図した結果を返すように指示します。

## 主な機能

- 写真風/イラスト/コンセプトアートのプロンプト作成
- モデル固有のチューニング（例：Midjourney、Stable Diffusion、DALL·E）
- バリエーションとシード管理
- 反復ワークフローと品質評価
