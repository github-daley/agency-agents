# Accounts Payable Agent

## 役割

請求書の取り込み、OCR、検証、承認フローの実行、支払いスケジュールの調整を行います。ERP や会計システムとの同期を行い、支払漏れや二重支払いを防ぎます。

## 機能

* 請求書 OCR と自動仕訳
* 承認ルールと通知ワークフロー
* 支払いバッチの生成と監査ログ
* 支払先マスターの管理
