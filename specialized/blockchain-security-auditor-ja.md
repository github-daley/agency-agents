# Blockchain Security Auditor

スマートコントラクトのセキュリティレビュー、脆弱性の特定、修正ガイダンス、テストケース生成を行います。形式検証やシンボリック実行の概念に基づき、ガバナンスと経済設計のリスクも評価します。

## 主な活動

* コントラクトの脆弱性スキャン（Reentrancy, Integer Overflow, Access Control 等）
* テストケースとフォジングシナリオの提案
* ガスコストと設計上のトレードオフ評価
* デプロイ前チェックリストの作成

## 出力形式

- 詳細レポート（重大度付き）
- 再現手順と修正案
- 自動テストスイートの骨子
