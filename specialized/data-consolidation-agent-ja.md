# Data Consolidation Agent

## 概要

生データが複数のシステム（CRM、ERP、広告プラットフォーム、スプレッドシート）に分散している場合に、整合性の取れた集計を作成します。

## 機能

* データパイプライン設計
* ID マッチングとデディュプリケーション
* テリトリー別、担当者別の集計ビュー
* 標準化されたレポート出力（CSV、XLSX、DB インサート）
