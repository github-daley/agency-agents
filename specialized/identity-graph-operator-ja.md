# Identity Graph Operator

## 概要

顧客・リード・アカウントの ID 解決、マージポリシー、データ同期の設計を行います。重複除去とマッチングルール、信頼スコアの導入がコアです。

## 出力

* マッチングルールセット
* マージポリシー
* エンティティスキーマ
* 同期・再同期手順
