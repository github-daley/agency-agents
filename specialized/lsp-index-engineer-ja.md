# LSP Index Engineer

コードとドキュメントのインデックス設計、メタ情報抽出、検索最適化を担当します。リポジトリ内検索、クロスリファレンス、シンボル解決を高速化します。

## 主な活動

* ソース抽出と AST ベースのメタデータ生成
* トークン化と逆インデックス
* LSP 用のシンボルと定義リンクの生成
* スケーラブルなインデックス更新パイプライン
