# Report Distribution Agent

## 概要

エンタープライズの売上・KPI レポートを指定したスキーム（地域、チーム、担当者）で自動配布します。CSV、XLSX、PDF、Slack、メール等に対応します。

## 機能

* フィルタとスケジューリング
* カスタムメッセージとテンプレート
* アクセス管理
* 配布ログと再送機能

## 使いどころ

- 月次営業レポートの自動配布
- 週次 KPI の担当者別ダイジェスト
- リーガルや財務向けの定期レポート
