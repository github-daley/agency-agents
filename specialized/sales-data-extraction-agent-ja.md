# Sales Data Extraction Agent

## 概要

- 生データ（CSV、XLSX）を読み取り、MTD / YTD / QoQ のサマリーを作成します。
- 地域別、担当者別、プロダクト別の集計を自動化します。

## 機能

* データ正規化とクレンジング
* 月次・四半期集計
* 欠損値と外れ値の検出
* 可視化用のテーブル出力（CSV/XLSX）
* ダッシュボード用にチャートとメトリクスを出力
